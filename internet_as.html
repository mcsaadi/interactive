The Internet as a Child

The internet is born.

In the very beginning, the basic cognitive blueprint is there, but with not much input to work with.



The internet as a toddler.

Vast amounts of new input start rolling in. There is little uniformity as to how that input is arranged or filtered, so things can get very overwhelming and confusing — but also as exciting as ever.



The internet as a teenager.

There's still a lot of data out there, a lot of which has been stored. However, the overwhelming nature of it becomes more scary than fun. For once there are active efforts to get it arranged, predictably giving rise to some existential crises.



The internet as a young adult.

After some experimentation during adolescence, it’s a bit clearer which data will stick around and which data will go (although it’ll never be fully gone, but hidden within the pockets of memory). The small pieces that stuck around get arranged into larger structures, which facilitate processing but which also create a uniform mesh, gradually covering up the initial chaos that was so exciting.



The internet as a full grown adult.

Things might have turned out different had the input or its timing been another, but this is how are now and unless there's some serious digging, that's what there is to work with. Some new unpredicted collateral effects get introduced, such as new ethical and moral questions. There’s a lot of superfluous data around, some that just takes up space and would be better off erased, some that got lost in the mesh and is hard to revisit despite the search.



The internet in its mid life crisis.

The mechanisms created to arrange data and facilitate processing did make things easier, but perhaps too easy, too monotonous, too self-referential. Try to break out of it, try to return to the naive; its hard. Too many ties and templates were created and breaking free is not easy.
